# Контекст проекта

Разделы (дополняйте и ведите `index` в каждой папке):

- **[[terminology|Термины]]** — сокращения и доменные термины.
- **[[technical-spec/index|Техническое задание]]** — требования, ограничения, сценарии.
- **[[meetings/index|Записи встреч]]** — kick-off, обсуждения экранов, решения.

## Назначение

ТЗ, бизнес-требования, договорённости, расшифровки встреч. Карта кода и пути к файлам — в [[structure/index|structure]].

## Для агентов

Сводный индекс спек и правил: [[agent/index|agent/index]]. Перед реализацией фичи сверьтесь с контекстом; при расхождении с кодом зафиксируйте решение новой заметкой и отметьте в [[daily-changes/index|daily-changes]].
