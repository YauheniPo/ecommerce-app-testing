# Записи встреч

Краткие протоколы: дата, участники, решения, follow-up.

| Встреча | Содержание |
|---------|------------|
| *(черновик)* | Добавьте строки или ссылки на файлы `YYYY-MM-DD-тема.md`. |

## Для агентов

Одна встреча — один файл или одна строка в таблице со ссылкой на файл `YYYY-MM-DD-тема.md`.
