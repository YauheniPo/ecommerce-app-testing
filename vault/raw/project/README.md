# `vault/raw/project` — машинный снимок репозитория

Слой **raw**: сюда попадают **только** автоматически сгенерированные файлы (опора для сверки с остальным ``vault/``).

- **Не редактировать вручную** — при следующем запуске `just vault-snapshot` изменения затрутся.
- Агент читает снимок и **обновляет** остальной `vault/` (`structure/`, `context/`, `daily-changes/`, при необходимости `agent/`).
- Команда: `just vault-snapshot` или `python3 scripts/vault_project_snapshot.py` из корня репозитория.
