# Слой `raw`

- **`project/`** — снимок репозитория (версии, деревья, выдержки контрактов). Создаётся только скриптом `scripts/vault_project_snapshot.py`; **не править от руки**.
- Остальной каталог **`vault/`** — живая база знаний; после `just vault-snapshot` её можно привести к фактам по навыку **vault-repo-sync**.
