# Frontend

- **Стек:** React, TypeScript, Vite, Chakra UI + MUI, Formik, Yup. Тема и токены: `frontend/src/theme/`.
- **Вход:** `frontend/src/main.tsx`, маршрутизация и оболочка в `frontend/src/App.tsx`.
- **Страницы:** `frontend/src/pages/`.
- **Компоненты:** `frontend/src/components/`.
- **Состояние:** `frontend/src/context/` (глобальный контекст).
- **API:** `frontend/src/api/client.ts`, типы в `api/types.ts`.
- **Мок-данные (dev):** `frontend/src/mockDB/db.ts`.

При новых экранах: страница в `pages/`, при необходимости маршрут в `App.tsx`, стили через тему MUI.
