# Сервер (backend)

Соответствует каталогу **`backend/`** в репозитории.

- Вход: `backend/main.py`, приложение в `backend/app/`.
- Модели и API: `models.py`, `schemas.py`, `crud.py`, `main.py`.
- БД и миграции: `app/db.py`, `alembic/versions/`.

Детальный чеклист путей: [[structure/backend]].

## Для агентов

При новых эндпоинтах или смене контрактов API — обновите эту заметку и при необходимости `vault/context/technical-spec/`.
